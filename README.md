# First-Blood

### [2017년도 1학기 대구과학고등학교 프로그래밍 실습1 프로젝트]

슈팅 게임 (1, 2인용)

1) 2명의 플레이어가 기본 공격과 다양한 아이템을 사용하여 겨루는 게임

2) 연습을 위한 1인용 게임도 가능

3) 게임에 등장하는 아이템 설정 가능
